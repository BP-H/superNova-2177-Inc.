# SuperNova 2177 Inc. Nonprofit Governance Docs

This folder contains GitHub-editable Markdown copies of the current SuperNova 2177 Inc. nonprofit governance working documents.

These Markdown files are meant for collaboration, review, and version history. Keep signed, adopted, and executed originals in the official board records.

## Documents

- [Bylaws](bylaws.md) - Proposed bylaws for SuperNova 2177 Inc.
- [First Organizational Board Meeting Packet](board-meeting-packet.md) - Board meeting agenda, script, resolutions, and minutes template.
- [Exhibits, Forms, and Policies](exhibits-forms-policies.md) - Director consents, conflict disclosures, and governance policies.
- [Final Review Action Plan](final-review-action-plan.md) - Step-by-step live meeting preparation and follow-up checklist.

## Editing Notes

- Use pull requests or commit history to track changes.
- Do not treat these working copies as executed legal documents until the board formally approves them.
- Preserve signed forms, meeting notices, consents, and minutes outside GitHub as official records.
